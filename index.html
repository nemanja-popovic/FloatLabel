Float Label

A jQuery plugin that creates floating labels in input elements. Float Label allows you to quickly create labels that show more details about input element for use in web applications and websites. The Float Label is easily customizable by changing the less styles and provides flexibility with options.

Features

  • Straightforward implementation with simple options
  • Can run on as many input elements as needed
  • Input elements can have small icon on right or left side
  • Floated labels look and feel is easily customized through the small less file

Supported input types

  • text
  • password
  • number
  • tel
  • email
  • search
  • url

Examples

<input class="flabel" type="text" value=""
        data-label-text="No icon"  />
<input class="flabel" type="text" value=""
        data-label-text="Down"  data-label-direction="down"  />
<input class="flabel"  type="text" value=""
        data-label-text="Information" data-label-icon="info" />
<input  class="flabel" type="number" value=""
        data-label-text="Number you love" data-label-icon="heart"
        data-label-icon-side="left" />
<input  class="flabel" type="tel" value=""
        data-label-text="Phone number" data-label-icon="phone"
        data-label-icon-side="left" />
<input class="flabel" type="email" value=""
        data-label-text="Email address" data-label-icon="message" />
<input class="flabel" type="password" value=""
        data-label-text="Password" data-label-icon="lock" />
<input class="flabel" type="search" value=""
      data-label-text="Term you want to find" data-label-icon="internet"
          data-label-icon-side="left" />
<input class="flabel" type="url" value=""
        data-label-text="Page address" data-label-icon="link" />

Usage

  1. Download all files
    • dist/float-label.min.js
    • float-label.css
    • img folder
  2. Include them in your html file along with jQuery
    <link href="stylesheets/float-label.css" rel="stylesheet" />
<script src="https://code.jquery.com/jquery-2.1.1.min.js"></script>
<script src="javascripts/float-label.min.js"></script>
  3. Example HTML:
    <input class="flabel" type="url" value=""
        data-label-text="Page address" data-label-icon="link" />
  4. Init on document ready event:
    $(document).ready(function () {
    $('.flabel').FloatLabel();
});

Options

Label text
data-label-text=""
Label moving direction

Sets the position of the label when input element has some value. Default value: Up

data-label-direction="down"
Icon
data-label-icon=""
